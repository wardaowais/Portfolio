Online Art Gallery

This project is a comprehensive e-commerce platform with full functionality for both users and administrators, developed using PHP, MySQL, HTML, CSS, and JavaScript. Users can register, log in, browse products, manage their cart, and place orders. Administrators can manage users, riders, products, and orders through a dedicated admin panel. The platform uses secure authentication, session management, and robust error handling. The database schema includes tables for users, products, orders, and riders. This platform demonstrates effective integration of frontend and backend technologies to provide a seamless online shopping experience and efficient administrative management.
